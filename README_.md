# Dr. Musaad Alruwaili Website

Static website for Dr. Musaad Alruwaili's digital headquarters.

## Publish With GitHub Pages

Recommended repository name for a personal GitHub Pages site:

```text
stranger616.github.io
```

After uploading these files to that repository, the site will be available at:

```text
https://stranger616.github.io/
```

## Files

- `index.html` - website content
- `styles.css` - design system and responsive layout
- `script.js` - mobile navigation behavior
- `assets/` - profile and banner images
- `.nojekyll` - tells GitHub Pages to publish the static files directly

## Before Launch

- Replace placeholder email address in `index.html`.
- Add Google Scholar, ORCID, Scopus, and ResearchGate URLs.
- Replace citation and h-index placeholders when confirmed.
- Update domain settings if using a custom domain.
